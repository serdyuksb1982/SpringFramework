Spring Framework, по шагам, согласно курса Udemy.

	Продолжение изучения Spring:
   Spring Boot. Знакомство со Spring Boot, фреймворком упрощения создания приложения на основе Spring.
В основе приложения тот же проект Employees, работающий с MySQL. Автоматизация конфигурирования. 
Управление зависимостями в maven проекте. Встроенная поддержка WEB-сервера. Spring Initializr. Конфигурация
на основе JPA (Java Persistens API) - стандартной спецификации системы управления сохранени Java объектов в 
таблицы БД, где Hibernate реализует правила, описанные в JPA. В Spring Boot дефолтной реализацией JPA является
Hibernate. Создание REST API приложения, использующее стандартные HTTP-запросы (СRUD).
